This can be used for cloud syncing your notes, check here for a tutorial.

Tutorial
It looks like this is the first time you're using Codex.

Click Continue if you would like to see a short tutorial about how to use Codex.
Tutorial (1/3)
First, let's create a new Notebook.

Click the icon next to the "NOTEBOOKS" section on the sidebar on the left.
Then, type in a name for the notebook, like what class it's for. You can also set a custom color for the notebook.
Tutorial (2/3)
Now let's create a Page inside that notebook.

Click on the notebook you just created to view what's in it.

Right-click on the new notebook, and click "New Page" and give it a name.

This could be something like a section/chapter in the class.
Tutorial (3/3)
Now you have a notebook page where you can write down your notes. Click on the page to edit it.

To add a code block to the page, click the "Type..." menu on the editor toolbar and choose your desired programming language.
Or, you can type a macro such as [cpp] for C++ or [js] for JavaScript.
